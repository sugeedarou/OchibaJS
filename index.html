Leaf JS

What is Leaf JS?

Leaf JS is a simple animation library to create stunning animations using CSS3 and Javascript.

How does it work?

It works by triggering css animations for every element after a calculated delay.

How can I create such animations?

Just download LeafJS from here

And now?

  1. Bind your parent element to LeafJS: var leaf = new LeafJS(document.getElementById('title'));
  2. Init your elements / letters : leaf.initLetters(); or leaf.init()
  3. Animate them: leaf.animateLeafs(options);
  4. Options are: speed, timing, delay, animation, animationType, callback
